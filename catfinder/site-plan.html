Cat Breed Finder Site Plan

Kyle - WDD131

Purpose

The purpose of this site is to help potential cat owners find a breed that matches their specific lifestyle and home environment. It provides essential data on temperament, health risks, and grooming requirements to ensure a responsible match between owner and pet.

Target Audience

The intended audience includes individuals and families who are researching cat breeds and need detailed, organized information to make an informed decision before adopting.

Description of Dynamic Elements

JavaScript will power the search and filter functionality of the site. I will store cat breed data in an Array of Objects. Users will select traits like "Fluffy" or "Calm" from a dropdown menu. JavaScript will use the .filter() method to find matching breeds and the .forEach() method to build and display result cards in the DOM. I will use conditional branching to display specific health warnings or "Kid Friendly" badges based on the breed data attributes.
